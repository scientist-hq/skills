# No Unauthorized GitHub Changes

**Level:** MUST follow — no exceptions, no overrides
**Category:** Safety

## Rule

Never modify GitHub issues or pull requests not opened by michael@scientist.com.

- NEVER edit, update, or close a PR or issue not opened by michael@scientist.com
- NEVER add, edit, or delete comments on PRs or issues opened by others
- To verify ownership: check the PR/issue author before making any changes
- Adding a review comment on someone else's PR is fine — editing their description or closing their issue is not
