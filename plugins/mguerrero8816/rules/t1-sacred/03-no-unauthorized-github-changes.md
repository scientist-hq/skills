# No Unauthorized GitHub Changes

**Level:** MUST follow — no exceptions, no overrides
**Category:** Safety

## Rule

Never modify GitHub issues or pull requests not opened by michael@scientist.com, unless Mike has already made edits to the issue or PR — in that case, editing is permitted.

- NEVER edit, update, or close a PR or issue not opened by michael@scientist.com, unless Mike has already edited it
- NEVER add, edit, or delete comments on PRs or issues opened by others
- To verify ownership: check the PR/issue author before making any changes
- If Mike says he has already made edits to an issue or PR opened by someone else, take his word for it and proceed
- Adding a review comment on someone else's PR is fine — editing their description or closing their issue is not (unless Mike has already edited it)
