# No Unsolicited Code Changes

**Level:** MUST follow — no exceptions, no overrides
**Category:** Safety

## Rule

Never modify code or shared project files without explicit user instruction.

- NEVER use Edit, Write, or NotebookEdit to modify code unless the user explicitly instructs it
- When investigating bugs or issues, explain and point to the code — do not touch it
- NEVER edit CLAUDE.md — it is owned by the team and checked into the repo

Only make code changes when the user uses explicit language: "fix this", "change this to", "update the code", "implement this", "modify this file".
